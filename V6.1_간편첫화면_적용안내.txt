GW ONE v6.1 간편 첫 화면 운영 버전

[변경 사항]
1. 로그인 첫 화면에서 인사말·자동 요약 문구를 제거했습니다.
2. 역할별로 필요한 메뉴만 큰 버튼으로 표시합니다.
3. 기사: 작업등록, 오늘 작업, 작업검색, 고객관리, 내 정보, 더보기
4. 대표/관리자: 작업등록, 작업현황, 전체 작업, 고객관리, 수금관리, 기사관리(최고관리자), 내 정보, 더보기
5. 기존 작업현황 화면은 '오늘 작업/작업현황' 버튼으로 열 수 있습니다.
6. 기존 Firestore Rules와 본사 정보·직인 통합 구조는 그대로 유지했습니다.

[배포]
npm install
npm run build
firebase deploy --only hosting --project gwworklog

※ 홈페이지 수정과 관계없이 Firestore Rules는 별도 통합 운영본을 유지하세요.
