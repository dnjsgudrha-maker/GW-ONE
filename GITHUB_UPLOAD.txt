GW ONE v1.7 GitHub 배포 준비

npm install 필요: 새 폴더라면 필요
Firebase 규칙 재게시: 필요 없음

이번 버전 추가 내용
- GitHub Pages 자동 배포 설정
- main 브랜치에 올리면 자동 배포
- Firebase 환경변수를 GitHub Secrets로 안전하게 관리
- 저장소 이름에 맞춰 자동 경로 설정
- 모바일 홈 화면 설치 가능한 PWA 유지
- README 및 배포 안내 추가

주의
- .env 파일은 GitHub에 올리면 안 됩니다.
- .gitignore에 .env가 포함되어 있어 자동으로 제외됩니다.
- Firebase Authorized domains에 깃허브아이디.github.io를 추가해야 구글 로그인이 됩니다.

가장 쉬운 진행 순서
1. 이 폴더를 GitHub 저장소에 업로드
2. GitHub Secrets 6개 등록

4. Actions 배포 완료 확인
5. Firebase 승인 도메인 추가
6. 휴대폰에서 배포 주소 접속
