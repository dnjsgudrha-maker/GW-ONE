GW ONE v6.4.1 사진 표시 복구

원인
- 일부 기존 작업의 사진이 주소 문자열이 아니라 사진 객체 형태로 Firestore에 저장되어 있었습니다.
- 화면에서 객체를 그대로 이미지 주소로 사용해 깨진 사진 아이콘이 표시되었습니다.

수정 내용
- 기존 객체 형태 사진에서 Cloudinary URL을 자동 추출
- 신규/기존 사진 모두 동일한 주소 형식으로 표시
- 작업 상세, 수정 화면, 문서 사진 영역에 동일하게 적용
- 임시 blob 주소는 제외
- Firestore Rules 변경 없음

적용 방법
1. 기존 GitHub 프로젝트에 전체 파일을 덮어씁니다.
2. Commit 후 Push 합니다.
3. GitHub Actions 배포 완료 후 새로고침합니다.
