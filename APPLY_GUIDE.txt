GW ONE v5.0 현장 완성 버전

포함 기능
- 모든 문서 본사 상호·사업자번호·대표자·연락처·주소·본사 직인 고정
- 문서의 작업자 표기를 '담당기사'로 변경하고 실명 표시
- 방문시간 추가 및 작업 상세·문서 반영
- 신규 작업 방문시간 현재 시간 자동 입력
- 누수탐지 '작업내용에 반영' 버튼 복구
- 작성 중 작업 임시저장·이어서 작성 질문 제거
- 휴대폰 뒤로가기: 내부 화면에서는 현황으로 이동, 현황에서 종료 여부 확인
- 오늘 기사별 현황에서 기사 클릭 시 해당 기사 오늘 작업 모아보기
- 기사 오늘 작업 방문시간순 표시
- 최근 작업 날짜·방문시간 기준 최신순 정렬
- 작업일지 3단계의 결제 데이터 누락 방어 처리
- 최고관리자 기사 대신 작성
- 수금관리·수수료·실수령·사진·동영상·문서 저장 유지
- 작업 상태 입력 기능은 추가하지 않음

적용
1. 압축을 풉니다.
2. src, site, package.json, package-lock.json을 GW-ONE 폴더에 덮어씁니다.
3. GitHub Desktop:
   Summary: Release GW ONE v5 field completion
   Commit to main
   Push origin

Firestore 규칙은 다시 게시하지 않아도 됩니다.
실제 프로덕션 빌드 성공 확인했습니다.
