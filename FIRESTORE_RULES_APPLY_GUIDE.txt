GW ONE + 홈페이지 통합 Firestore Rules 적용 방법

1. Firebase Console 접속
2. Firestore Database → 규칙
3. 기존 규칙 전체 삭제
4. 프로젝트의 firestore.rules 전체 복사 후 붙여넣기
5. "게시" 클릭
6. 모바일 GW ONE 완전 종료 후 다시 실행

권한 구조
- 홈페이지 설정/후기: 공개 조회, 최고관리자 관리
- 기사: 본인 작업 조회·등록·수정·삭제
- 대표: 전체 작업 조회, 수금 처리, 작업자 사전등록
- 최고관리자: 전체 작업/정산/사용자/관리자 권한 관리
- 전체 작업현황: collectionGroup("jobs") 조회 허용
