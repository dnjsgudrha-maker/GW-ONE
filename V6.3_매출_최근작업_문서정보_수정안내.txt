GW ONE v6.3 수정 내용

1. 첫 화면 매출 표시
- 오늘 작업 건수
- 오늘 매출
- 이번 달 작업 건수
- 이번 달 매출
- 대표/관리자는 전체 작업 기준, 기사는 본인 작업 기준

2. 작업현황 최근 작업
- 다른 담당기사 이름을 프로필 정보와 배정 기사 정보로 표시
- 기사 선택 목록에 전체 사용자 표시
- 최근 작업 정렬을 작업일/방문시간순이 아닌 등록순(createdAt 최신순)으로 변경
- createdAt이 없는 과거 작업은 updatedAt, 작업일/시간 순으로 보완

3. 문서 회사정보 정리
- 회사명 유지
- 대표자명 유지
- 사업자등록번호 유지
- 본사 직인 유지
- 담당기사 이름 유지
- 회사 전화번호, 이메일, 주소 제거

4. 후기요청 버튼
- v6.2 수정 내용 그대로 유지

5. Firestore Rules
- 변경 없음
