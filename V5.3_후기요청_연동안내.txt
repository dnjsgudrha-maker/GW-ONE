GW ONE v5.3 후기요청 기능

1. .env 파일에 홈페이지 주소를 입력합니다.
   VITE_HOMEPAGE_URL=https://실제홈페이지주소

2. 작업 상세의 [후기 요청 보내기] 버튼을 누르면
   - 작업일지에 후기요청일과 전용 링크가 저장됩니다.
   - 휴대폰 문자 앱이 고객 전화번호와 문구가 입력된 상태로 열립니다.
   - 사용자가 문자 앱에서 최종 전송을 누릅니다.

3. 문자 문구
   후기 작성 후 담당 기사에게 ‘후기 작성 완료’라고 문자 보내주시면,
   커피쿠폰 또는 소정의 사은품을 보내드립니다.

4. 홈페이지 후기 등록 시 아래 값을 homepageReviews 문서에 함께 저장해야 합니다.
   const params = new URLSearchParams(window.location.search);
   const jobId = params.get('jobId') || '';
   const ownerUid = params.get('ownerUid') || '';

   후기 저장 데이터에 다음 두 필드를 추가:
   jobId,
   ownerUid,

5. 홈페이지에 위 두 필드가 저장되면 GW ONE에서 자동으로
   [후기작성완료] 상태가 표시됩니다.

6. 사은품 전달 후 [사은품 전달완료 처리]를 누르면 처리일이 작업일지에 저장됩니다.

주의:
웹브라우저/PWA에서 사용자의 확인 없이 문자를 완전 자동 발송할 수는 없습니다.
현재 버전은 문자 앱을 자동으로 열고 고객번호와 문구를 미리 입력합니다.
완전 자동 발송은 별도의 문자 API와 서버 비밀키 설정이 필요합니다.
